# Bhumiq Pro Calculator

Bhumiq Pro is a premium calculator web app with a clean and polished design. It is built using HTML, CSS, and JavaScript.

## Features

- Standard calculator functions: addition, subtraction, multiplication, and division
- Responsive layout for desktop and mobile screens
- Copy button to copy the result to the clipboard
- Calculation history with last entries visible
- Keyboard support for numbers and operators
- Friendly user messages and visual effects for a more engaging experience

## How to use

1. Open `index.html` in a web browser.
2. Use the buttons or keyboard to enter numbers and operators.
3. Press `=` or Enter to calculate the result.
4. Press `COPY` or `C` to copy the result to the clipboard.
